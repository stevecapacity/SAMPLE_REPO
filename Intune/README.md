# Intune Scripts

Scripts in this folder focus on Microsoft Intune device management and reporting.

## Common Use Cases

- Device compliance reporting
- Enrollment validation
- User and device association
- Operational visibility gaps in the Intune portal

## Notes

- Scripts in this folder are read-only unless explicitly stated.
- Most scripts require `DeviceManagementManagedDevices.Read.All`.

Each script includes usage examples and permission requirements at the top.
