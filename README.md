# GetRubix PowerShell Scripts

This repository contains practical PowerShell scripts for Microsoft Intune, Entra ID, and Microsoft 365 administration.

These scripts are written from a real-world IT admin perspective. They focus on:
- Readability over cleverness
- Safe defaults
- Clear documentation
- Reusability across tenants

## Repository Structure

- **Intune**  
  Scripts for device management, compliance, and reporting.

- **EntraID**  
  Scripts for identity, users, groups, and access visibility.

- **Utilities**  
  Helper scripts used across multiple scenarios.

- **Templates**  
  Reusable templates for writing consistent PowerShell scripts.

## Requirements

- PowerShell 7 recommended
- Microsoft.Graph PowerShell module
- Appropriate Microsoft Graph permissions per script

## Disclaimer

Always test scripts in a non-production tenant or with test data before using them in production.
